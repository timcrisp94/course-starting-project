Core Types

  • Number: all numbers, same as JS
  • String: all text values, same as JS
  • Boolean: true, false; no truthy falsy values
  • Object: any JS object, and more specific types

TypeScript does not change run-time js, it helps us through development

--TypeScript is static while JS is dynamic

Assigning types

Type inference: TS does its best to understand type by looking at the variable assignment. If a variable is assigned a value, TS gets it. However, if you declare a variable without setting a value, it is good practice to assign a type. For example:

let number1: number
